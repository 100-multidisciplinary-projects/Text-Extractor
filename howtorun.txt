Tamil OCR Application - Guide
Overview
A powerful command-line application for extracting Tamil text from images using advanced OCR engines and AI validation. The system combines multiple preprocessing techniques with state-of-the-art language models to achieve high-accuracy Tamil text recognition.

Features

Dual OCR Engine Support: Choose between ocr_tamil and PaddleOCR
5 Preprocessing Methods: Optimize for different image types
AI Validation: Qwen2.5 72B model corrects OCR errors
Batch Processing: Process multiple images simultaneously
Confidence Assessment: Automatic quality evaluation
Detailed Outputs: Save preprocessed images and results

#For translation of three languages: Chinese, Japanese, French:
Run in terminal:
python translate.py image.jpg --api-key YOUR OPENROUTER KEY --ai --translate

# Basic usage
python app.py image.jpg

# Use PaddleOCR with auto preprocessing
python app.py image.jpg --engine paddleocr --mode auto

# Enable AI validation
python app.py image.jpg --api-key YOUR_OPENROUTER_KEY --ai

# Process multiple images with detailed output
python app.py img1.jpg img2.jpg img3.jpg --details --output ./results

# Specific preprocessing method
python app.py image.jpg --mode standard --engine ocr_tamil

# Full featured command
python app.py image.jpg --engine paddleocr --mode auto --api-key YOUR_KEY --ai --details --output ./my_results
